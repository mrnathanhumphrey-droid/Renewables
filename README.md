# Renewables

A research repository for renewables-improvement work. Currently one substrate is active:

- **[Battery/](Battery/)** — multi-operator disagreement-onset framework for battery aging.
  Phase 0 → Phase 4 closed; Phase 4 multi-cohort verdict landed across SECL, Khan 2025,
  Zhang Cambridge v2; C1 cross-chemistry hierarchical pooling now extended to 4
  chemistry-form-factor groups (NMC_cyl, NMC_prism, LCO_button, NMC811_cyl).

See each substrate's own README for the actual scientific content, methods, and current state.

## Scope

This repo is a research log, not a deployed product. Code is the substrate for
analysis; results are checkpointed in the `literature/` folder of each substrate
as Markdown decision memos. Commits are the canonical history.
